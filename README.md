# 欢迎来到PlayFun-FateAlpha(ARPG)
# 这是一个持续开发,并且功能健全的完全开源跨平台游戏项目(iOS/Android/Win32等)

* 这个项目的初衷是为了将WOW模拟器中学习到的知识应用到新的领域.并且一步一步的完成独立游戏的开发制作!
* 这意味着我将专注于:
	* 完全独立开发.并且注重玩家体验
* 所做的事情:
	* 这个项目非常期待大家的反馈
	* 此外,项目本身需要完善的地方还有很多.希望大家能一起合作开发
	* 欢迎任何喜欢,或者有志与共同开发,探讨的朋友一起加入！

	* World-Loading(纯读取)

	* Manager
		* NotifyMgr(消息提示)
		* QuestMgr(任务)
		* SpellMgr(技能)
		* DataMgr(数据)
		* ScriptMgr(脚本)
		* MovementMgr(移动)

			* Monster
